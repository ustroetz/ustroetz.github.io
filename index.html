Logging Road Model




The tool models logging roads based on existing OSM street data, forest stands, and a cost surface.

A project for Ecotrust's Forest Planner.

Timber Harvest Cost Model




The model estimates total delivered costs ($ US) for harvesting a set of timber stand from the stumpage to the mill gate.

A project for Ecotrust's Forest Planner.

Bike Garage




Bike Garage helps you find the closest bike rake to you.

A project with maptimePDX.

Biomass Potential Analysis for Brandenburg, Germany




For the research project BIOBRA a GIS analysis was conducted about what amount of currently unused biomass in the state of Brandenburg is available. The focus was one organic household waste, sludge, biogas remains, spent grains.

A project for HNEE's project BIOBRA.
